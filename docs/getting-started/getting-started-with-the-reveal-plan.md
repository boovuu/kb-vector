# Getting Started with the Reveal Plan

> Source: https://learn.vector.co/articles/1775877825-getting-started-with-the-reveal-plan

---

The Vector Reveal Plan is designed for teams that want to identify and act on website visitor intent without the full suite of CRM and ad platform integrations.

## What's Included in the Reveal Plan
- Vector Pixel installation
- Visitor Feed with contact-level identification
- Segment creation (up to 5 segments)
- Slack notifications
- CSV export of visitor data
- Account List uploads
- Basic webhook support

## What's Not Included (Available on Higher Plans)
- Native CRM integrations (HubSpot, Salesforce)
- Native ad platform integrations (LinkedIn, Google, Meta, etc.)
- Campaign Reporting dashboard
- Advanced segment logic
- Unlimited segments
- SSO

## Getting Started Steps

1. **Install the Pixel**: Add the Vector Pixel to your website (Settings → Pixel → copy snippet)

2. **Verify Installation**: Check Settings → Pixel to confirm it shows "Active"

3. **Upload Target Accounts**: Add your target company list (Settings → Account Lists)

4. **Create Your First Segment**: Build an ICP segment to filter for your best-fit visitors

5. **Connect Slack**: Set up Slack notifications so your team gets alerted when qualified visitors appear

6. **Review Daily**: Check the Visitor Feed daily for the first few weeks to understand what's working

## Upgrading
To access CRM integrations, ad platform syncing, and Campaign Reporting, contact sales@vector.co to explore higher-tier plans.
