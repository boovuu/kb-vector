# Getting Started Checklist

> Source: https://learn.vector.co/articles/9573423468-getting-started-checklist

---

Use this checklist to get Vector set up and capturing intent data quickly.

## Week 1: Installation & Verification

- [ ] Install the Vector Pixel on your website (see Pixel installation guide)
- [ ] Verify the pixel is firing: Settings → Pixel → should show "Active"
- [ ] Add your company domain to the Denylist to exclude internal traffic
- [ ] Invite your team members (Settings → Team)

## Week 1-2: First Segments

- [ ] Create your first segment (e.g., "ICP Visitors" — contacts who match your ideal customer profile)
- [ ] Set up a Slack notification for your segment so your team gets alerted in real time
- [ ] Review your Visitor Feed daily for the first week to understand who's visiting

## Week 2-3: CRM Integration

- [ ] Connect HubSpot or Salesforce (Integrations → your CRM)
- [ ] Verify that timeline events are appearing on contact records in your CRM
- [ ] Set up lead scoring rules using Vector properties

## Week 3-4: Ad Platform Connections

- [ ] Connect LinkedIn Ads (highest match rates for B2B)
- [ ] Connect Google Ads
- [ ] Enable audience syncing for your key segments
- [ ] Launch retargeting campaigns using Vector audiences

## Ongoing

- [ ] Review and refine segment criteria based on conversion data
- [ ] Check pixel status weekly
- [ ] Upload updated target account lists monthly
