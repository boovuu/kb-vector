# Settings Overview

> Source: https://learn.vector.co/articles/7714116747-settings-overview

---

The Settings area in Vector is where you manage your account configuration, pixel setup, integrations, and team.

## Settings Sections

### General
- Account name and timezone
- Notification preferences
- Default segment settings

### Pixel
- View your Pixel ID and installation snippet
- Check pixel status (active/inactive)
- View recent pixel events for debugging

### Team
- Invite team members
- Manage roles (Admin, Member)
- Remove users

### Integrations
- Connect CRM platforms (HubSpot, Salesforce)
- Configure webhooks
- Manage ad platform connections (LinkedIn, Google, Meta, etc.)
- Set up Slack notifications

### Allowlist / Denylist
- Exclude specific IP addresses, email domains, or companies
- Ensure only relevant traffic appears in your feed

### Billing
- View your current plan
- Manage payment methods
- View invoice history

### API
- Generate and manage API keys
- View API documentation links

### SSO
- Configure SAML 2.0 single sign-on
- Manage SSO provider settings
